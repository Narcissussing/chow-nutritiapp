# chow-nutritiapp
Application web affichant les informations nutritionnelles des aliments courants.

## Objectif
Apprendre à structurer un projet multi-pages en HTML/CSS/JavaScript,
consommer des données depuis un fichier JSON simulant une API,
et pratiquer le versioning avec Git et GitHub.

## Aliments couverts
20 aliments incluant légumes, fruits, viandes, céréales et produits laitiers.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript (Vanilla)
- JSON (simulation d'API)
